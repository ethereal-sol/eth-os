# eth-os: ethereal open-source project

eth-os — это открытый (opensource) проект, созданный для **приватных серверов** (клонов) популярной игры **Standoff 2**.

Проект включает технологию **автообновления** и обеспечивает **нативную поддержку архитектуры ARM64** (с возможностью адаптации под ARM32).

### ✨ Основные особенности:
* **Технология автообновления** (Autoupdate).
* **Высококачественный код:** Чистый и хорошо структурированный синтаксис для комфортного чтения и разработки.
* **Игровая эффективность:** Включает продвинутый **ESP** и лучший в классе **Silent Aim**.
* **Интерфейс:** Красивые, настраиваемые **анимации imgui с физикой**.
* **Отладка:** Встроенное **отладочное меню (Debug Menu)**.
* **Стабильность:** Включена **защита от критических сбоев**, хотя код находится на ранней стадии разработки и может требовать доработки.

### ⚠️ Совместимость
Не работает с серверами: *Standleo*, *Standchillow 1.6* и *2.0*.
